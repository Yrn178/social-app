# SparQ - Социальная сеть

Учебный проект социальной сети с возможностью создания постов, лайков, комментариев и загрузки изображений.

## Требования

- Node.js (версия 18 или выше)
- npm (устанавливается вместе с Node.js)

## Как установить

### 1. Клонирование репозитория

```bash
git clone https://github.com/Yrn178/social-app.git
cd social-app
```
### 2. Установка зависимостей
Фронтенд:
```bash
npm install
```
Бэкенд:
```bash
cd server
npm install
cd ..
```
### 3. Запуск приложения
Терминал 1 - Бэкенд:
```bash
cd server
node index.js
```
Ожидаемый вывод: Server running on http://localhost:3001
Терминал 2 - Фронтенд:
```bash
npm run dev
```
Откройте http://localhost:5173 в браузере
